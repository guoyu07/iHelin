# [Ian He](https://www.ianhe.me)

## 前言
`iHelin`项目是一套小型的开源博客系统，包含文件上传，二维码生成等工具。


## 项目简介

### 技术选型

#### 后端技术

- Spring Framework
- Spring Security
- Spring MVC: MVC框架
- MyBatis: ORM框架
- Druid: 数据库连接池
- Logback: 日志管理
- Maven: 项目构建管理
- Redis: 缓存服务
- ActiveMQ: 消息队列

#### 前端技术

- Vue: 前端渐进式框架
- jQuery
- Bootstrap
- Font-awesome
- ...

> 项目正准备前后端分离

### 环境搭建
#### 开发工具

- IntelliJ IDEA
- Git
- Nginx: 反向代理服务器
- Navicat for MySQL & MySQL Workbench: 数据库客户端

#### 开发环境
- Java 1.8.0_111
- Mysql 5.6.35
